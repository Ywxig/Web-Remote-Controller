Немного об мне

Я веб-приложение на Flask для удалённого управления ПК на windows. Я могу помочь совершать сёрфинг в интернете, легко и просто включать и выключать видео на устройстве.

Во сановном можете использовать для быстрого включения видео на ютубе. Мой интерфейс нацелен на удобство и простату. Также я собираю некоторые данные, такие как:

  • Читаю ваши запросы для организации ленты устройства. Благодаря этому вы сможете выбирать видео, которые я найду на просторе ютуба
  • Я знаю о перемещении мышь. Я запоминаю движение мыши по экрану устройства, это необходимо для вашего опыта использование

Пожалуйста, если вам понравился Web-Remote-Controller, то посетите GitHub разработчика, он закреплён в меню ❤❤❤
